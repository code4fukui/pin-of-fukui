# pin-of-fukui
[README.ja.md](README.ja.md)にある日本語のREADMEをご覧ください。

福井県の場所情報を使って作成したマップです。

## デモ
[ライブデモ](https://code4fukui.github.io/pin-of-fukui/)

## 機能
- 位置データを持つピンの地図を表示します
- グレースケールモードをサポートしています
- 各ピンのポップアップ情報を含んでいます
- [福井のPin!](https://makef.jp/pin/)からのオープンデータを利用しています

## 要件
このプロジェクトには、最新のウェブブラウザが必要です。

## 使用方法
1. [live demo](https://code4fukui.github.io/pin-of-fukui/)をウェブブラウザで開きます。
2. マップと連携して、ピンの情報を確認します。

## データ/ API

このプロジェクトのデータは、福井県の様々な場所に関するオープンデータを提供している[福井のPin!](https://makef.jp/pin/)のウェブサイトから得られています。
